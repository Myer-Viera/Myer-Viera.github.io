The Basics of Vim

Vim is one of the oldest coding languages in the world, despite this it is stil widely used. Today I am going to teach you the basics of vim.

Moving around The h key moves left The l key moves right The j key moves down The k key moves up Remember that you can also use the arrow keys for the same purpose. Exiting 1. Press the escape key (to make sure you are in Normal mode). 2. Type the command :wq and then press the ENTER key. This exits the editor,saving any changes you have made. 3. Get back here by executing the command that got you in. That might be: vim terminal.html Text Editing Deletion 1. To fix the errors, move the cursor until it is on top of the character to be deleted. 2. Press the x key to delete the unwanted character. 3. Repeat steps 1 and 2 until the sentence is correct. Insertion 1. To edit an already made line, move the cursor on top of the character BEFORE where you want to type. 3. Press i and type in the necessary additions. 4. When you are finnished adding in what you need, make sure to press the escape key to get back into normal mode.
